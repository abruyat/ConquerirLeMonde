Episode 1 : Vagrant and CoreOs

 Copy files from usb key 
      
        cp USB/coreos_production_vagrant.box coreos_production_vagrant.box
Create config file 
      
        touch Vagrantfile
This file contains 
        Vagrant.configure("2") do |config|
        
            config.vm.box = "coreos-alpha"
        
            config.vm.network "private_network", ip: "192.168.33.10"
        
            config.vm.network "public_network
        
        end
Create vagrant box 
      
        vagrant box add coreos-alpha coreos_production_vagrant.box
Start your box 
      
        vagrant up
Connect to your box 
      
        vagrant ssh
