Episode 7 : Let's kill them all to dominate

 Download and install ansible 
      
        http://docs.ansible.com/ansible/intro_installation.html#installing-the-control-machine
Modify Host file 
      
        vi /etc/ansible/hosts
Add all planets in your file 
      
        [all:vars]
        ansible_connection=ssh 
        ansible_user=jarjar 
        ansible_pass=binks
        
        [planets]
        #PLANET_NAME ansible_host=#PLANET_IP ansible_port=22 ansible_user=jarjar
Ping all planets 
      
         ansible all -m ping
View all docker 
      
        ansible all -s -m shell -a 'docker ps'
Kill all docker 
      
        ansible all -s -m shell -a 'docker kill #TROOPER_NAME'
