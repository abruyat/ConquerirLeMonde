Create nginxfile-service.json file 
      
        touch nginxfile-service.json
This file contains: 
      
           
        {{"{"}}
            "kind":"Service",
            "apiVersion":"v1",
            {{"\"metadata\":{"}}
                "name":"nginxfile",
                {{"\"labels\":{"}}
                    "app":"nginxfile"
                {{"}"}}
            {{"},"}}
            {{"\"spec\":{"}}
                "ports": [
                {{"{"}}
                    "port":80,
                    "targetPort":"http-server",
                    "nodePort": 32100
                {{"}"}}
                ],
                {{"\"selector\":{"}}
                    "app":"nginxfile"
                {{"},"}}
                "type": "LoadBalancer"
            {{"\}"}}
        {{"\}"}}
Run replicas 
      
        ./kubectl create -f nginxfile-controller.json
View Replicas 
      
        ./kubectl get rc
View pods 
      
        ./kubectl get pods
Run service 
      
        ./kubectl create -f nginxfile-service.json
View Services 
      
        ./kubectl get services
Connect to your nginx service 
      
        curl -XGET http://YOUR_IP:32100/
